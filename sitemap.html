🔄 User Flow
ขั้นตอนผู้ใช้หน้าจอการกระทำผลลัพธ์
1 พนักงาน pet-donate.html เปิดหน้า → เห็น Hero Stats ยอดรวมทั้งบริษัท รับรู้ผลกระทบรวมของทีม
2 พนักงาน pet-donate.html กรอกชื่อ รหัส แผนก เลือกจำนวนขวด เห็น Preview CO₂ ที่จะลดได้ทันที
3 พนักงาน pet-donate.html กด "บันทึกการบริจาค" ข้อมูลบันทึกใน localStorage + แสดง Success
4 พนักงาน pet-donate.html ดู Leaderboard เห็นอันดับ + ขวด + กก.CO₂e ของตัวเองและคนอื่น
5 Admin pet-admin.html เปิด Dashboard → กรองช่วงเวลา/แผนก เห็นสถิติ GHG และกราฟอัปเดตอัตโนมัติ
6 Admin pet-admin.html กด Export CSV ดาวน์โหลดรายงานทุกรายการ
